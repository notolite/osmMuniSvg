## usage
```bash
# for code, input R357794 for Kyoto city for example,
# whose url is https://www.openstreetmap.org/relation/357794.

python main.py <code>
```
## example work
You can edit the outcome svg code to customise the image no matter how you like to.
Here is an example: Yokohama city coloured by the keikenchi (visit status) of each ward, where red denotes staying, yellow denotes walking and blue denotes passing while riding something.
[https://notolite.github.io/keikenchi/yokohama.svg]
